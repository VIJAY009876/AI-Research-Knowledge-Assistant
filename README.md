# AI-Research-Knowledge-Assistant
**Problem** : 
Researchers and Students Spend hours :
 - Reading long research paper , Finding relevent information , Comparing multiple papers Creating notes, remembering Previous research.

**Solution**:
Build Ai Assistant that can :
 - Read reasearch Paper , compare multiple paper , prepare notes , Cite         sources,Help researchers Work faster and Ans Qustions.

Building an end-to-end AI Research & Knowledge Assistant using: 
- LLMs 
- RAG 
- LangChain 
- LangGraph 
- AI Agents 
- MCP 
This project is developed incrementally as part of a GenAI Bootcamp.
